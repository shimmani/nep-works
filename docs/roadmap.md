# 개발 로드맵

## Phase 1: 기반 + 프로젝트 관리 + 기성처리 ✅

**기간**: 8~10주 | **상태**: 구현 완료 + 개인화 강화

| 주차 | 작업 | 상태 |
|------|------|------|
| 1-2 | Electron + React + TypeScript 프로젝트 세팅, SQLite 연동, DB 스키마 | ✅ |
| 3 | 공통 UI (레이아웃, 사이드바, 테이블, 폼) | ✅ |
| 4-5 | 프로젝트 CRUD, 발주처 관리, 상태 관리 | ✅ |
| 5-6 | 엑셀 엔진 (ExcelJS 기반 읽기/쓰기) | ✅ |
| 7-8 | 기성처리: 설계내역서 임포트, 회차 관리, 진도율→금액 자동계산 | ✅ |
| 9-10 | 기성 엑셀 출력, 증빙서류 체크리스트 | ✅ (엑셀 출력 완료, 체크리스트 UI 예정) |
| + | 슈퍼 개인화: 감사 로그, 비즈니스 검증, 워크플로우 자동화, 추천 엔진 | ✅ |
| + | 휴먼 체크포인트: 프리뷰/확인 모달, 경고 표시, 변경이력 탭 | ✅ |
| + | 테스트: 300개 케이스 (validation 100 + audit/workflow 100 + integration 100) | ✅ |

**완료 기준**: 실제 프로젝트 1건으로 기성내역서 생성 가능

## Phase 2: 준공서류 + 노무비 (6~8주)

| 주차 | 작업 |
|------|------|
| 1-2 | 준공서류 체크리스트 (발주처별 템플릿) |
| 3-4 | 준공사진첩 자동 생성 (EXIF 추출, 공종 분류, 엑셀/PDF 출력) |
| 5-6 | 근로자 등록, 출역 달력 |
| 7-8 | 노무비 산출 (4대보험/원천징수), 지급대장 엑셀 생성 |

**완료 기준**: 준공사진첩 자동 생성, 월별 노무비 명세서 생성 가능

---

## Phase 3: 입찰/계약 + 외부 연동 (6~8주)

| 주차 | 작업 |
|------|------|
| 1-2 | 나라장터 API 연동 (공고 조회, 필터) |
| 3-4 | 입찰/수의계약 관리, 계약→프로젝트 전환 |
| 5-6 | 홈택스 CSV 생성, 4대보험 EDI 파일 |
| 7-8 | 대시보드, 시스템 트레이 알림 |

---

## Phase 4: 고도화 (4~6주)

- 템플릿 매핑 시각적 UI
- 자동 백업/복원
- 사업계획서·시공계획서 생성
- 통계/수익성 분석
- Windows 인스톨러 패키징

---

## 외부 연동 계획

| 시스템 | 방식 | 시기 |
|--------|------|------|
| 나라장터 | REST API (data.go.kr 무료) | Phase 3 |
| 홈택스 | CSV 파일 생성 → 수동 업로드 | Phase 3 |
| 4대보험 | EDI 파일 생성 → 각 공단 업로드 | Phase 3 |
| 건설근로자공제회 | CSV 생성 → 수동 업로드 | Phase 4 |
