# 기능 명세

## Phase 1 (현재 구현 완료)

### 1. 대시보드
- 진행중 프로젝트 수, 시공중 건수, 준공 예정 건수, 총 계약금액 통계
- 진행중 프로젝트 목록 테이블

### 2. 프로젝트 관리
- **CRUD**: 프로젝트 등록/수정/삭제
- **상태 관리**: 입찰중 → 계약체결 → 착공 → 시공중 → 준공서류작성 → 준공검사 → 준공완료 → 하자보증 → 완료
- **프로젝트 상세**: 기본정보, 설계내역, 기성 탭
- **필드**: 공사명, 발주처, 계약유형(종합/전문/일반/용역), 계약방식(입찰/수의계약), 계약금액, 계약기간

### 3. 발주처 관리
- **CRUD**: 발주처(지자체) 등록/수정/삭제
- **필드**: 발주처명, 지역, 담당자, 연락처, 주소
- **삭제 보호**: 프로젝트가 있는 발주처는 삭제 불가

### 4. 설계내역 임포트
- **엑셀 자동 파싱**: 설계내역서 엑셀 파일을 업로드하면 자동으로 항목 추출
- **헤더 자동 탐지**: '항목명', '수량', '단가' 등 키워드로 헤더 행 자동 탐색
- **원가 구분 자동 추정**: 공종명/항목명에서 재료비/노무비/경비 자동 분류
- **보호**: 기성이 진행된 프로젝트는 설계내역 재임포트 차단

### 5. 기성처리
- **회차 자동 생성**: 설계내역 기반으로 기성 상세 자동 생성
- **전회 누계 자동 이월**: 이전 회차의 누계가 다음 회차의 전회로 자동 설정
- **진도율 입력**: 공종별 금회 진도율 입력 → 금액 자동 산출
- **자동 계산**: 전회/금회/누계 금액·비율 자동. 회차 총 기성금액 자동 합산
- **100% 초과 방지**: 누계 진도율이 100%를 초과하면 에러
- **상태 관리**: 작성중 → 청구완료 → 승인완료 / 보완요청

### 6. 기성내역서 엑셀 내보내기
- **양식**: 공사명, 발주처, 도급금액, 기성일자 헤더 포함
- **내역**: 공종별 설계금액, 전회/금회/누계(비율+금액) 표시
- **서식**: 셀 테두리, 배경색, 숫자포맷(#,##0), 퍼센트포맷, 병합셀
- **합계행**: 설계총액, 전회/금회/누계 합계 및 비율 자동 계산

### 7. 설정
- 보험요율 정보 표시 (2026년 기준)

### 8. 슈퍼 개인화 (Phase 1.5)

#### 감사 로그 (변경이력)
- 모든 엔티티의 생성/수정/삭제 자동 기록
- 필드별 변경 추적 (이전값 → 새값)
- 프로젝트 상세 "변경이력" 탭에서 전체 이력 조회

#### 비즈니스 검증
- **프로젝트**: 필수값, 수의계약 한도(2026년 기준), 날짜 논리 검증
- **상태전이**: 유효 상태 머신 검증 (예: 시공중→준공서류작성 가능, 입찰중→시공중 불가)
- **기성 진도율**: 범위(0~100%), 누계 100% 초과 방지, 대규모 첫회차 경고
- **설계내역**: 계약금액 대비 비율 경고, 0원/음수 항목 경고
- **엑셀 내보내기**: 전체 0% 경고, 0원 기성 경고
- errors(차단) + warnings(사용자 확인 후 진행) 분리

#### 휴먼 체크포인트
- **설계내역 임포트 프리뷰**: 파싱 결과 미리보기 (항목 수, 금액, 공종별/비용구분별 분석, 검증 경고) → 확인 후 임포트
- **기성 회차 생성 프리뷰**: 다음 회차 번호, 잔여금액, 진행률, 기존 회차 현황 → 확인 후 생성
- **엑셀 내보내기 프리뷰**: 6개 통계(설계금액, 금회, 누계, 진행률, 항목 수, 변경 항목) + 검증 경고 → 확인 후 내보내기
- **프로젝트 저장 검증**: 경고 발생시 Modal.confirm으로 사용자 확인

#### 워크플로우 자동화
- 프로젝트 상태 변경시 할일 자동 생성 (계약체결/시공중/준공서류작성/준공완료)
- 대시보드에 전체 미완료 할일 목록 표시
- 프로젝트 상세 "할일" 탭에서 프로젝트별 할일 관리 (완료/건너뛰기)
- 다음 단계 추천 (현재 상태 + 데이터 완성도 기반)

#### 추천 엔진
- 발주처 선택시 과거 프로젝트 기반 기본값 추천 (계약유형, 방식, 평균 금액)
- 기성 진도율: 같은 발주처의 과거 프로젝트에서 동일 회차 평균 진도율 추천
- 발주처별 커스텀 기본값 저장/관리

## Phase 2 (구현 완료 — 사진첩 제외)

### 9. 준공서류 체크리스트
- **기본 템플릿**: 12항목 (준공계, 준공내역서, 준공사진첩, 품질시험성과총괄표, 자재검수확인서, 안전관리비사용내역, 환경관리비사용내역, 하자보증서, 하도급관련서류, 폐기물처리확인서, 노무비지급확인서, 4대보험 납부확인서)
- **상태 관리**: 미완료 → 완료 / 해당없음
- **파일 연결**: 각 항목에 파일 경로 첨부
- **진행률**: 전체/완료/미완료/해당없음 통계 + Progress 바
- **엑셀 내보내기**: 체크리스트 엑셀 생성 (완료 항목 녹색 배경)

### 10. 일용직 노무비 관리
- **근로자 CRUD**: 등록/수정/삭제/비활성화, 직종별 관리
- **출역 관리**: 프로젝트×월별 출역 조회, 단건/일괄 등록, 전일 복사
- **급여 자동 계산**:
  - 총액 = Σ(일당 × 일수)
  - 국민연금 4.5%, 건강보험 3.545%, 장기요양 = 건강보험 × 12.95%, 고용보험 0.9%
  - 일용근로소득세: 일별 (일당 - 15만원) × 2.7% (6% × 45%)
  - 지방소득세: 소득세 × 10%
- **노무비 지급대장 엑셀**: 근로자별 출역일수, 총액, 공제내역, 실지급액, 은행/계좌
- **비즈니스 검증**: 근로자/출역/급여계산/급여내보내기 각각 errors + warnings 분리

### 미구현 (추후)
- 준공사진첩 자동 생성 (EXIF 날짜 추출, 공종별 분류, 양식 자동 배치)
- 준공내역서 (최종 기성 기반)
- 일괄 패키지 폴더 정리
