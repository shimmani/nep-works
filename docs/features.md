# 기능 명세

## Phase 1 (현재 구현 완료)

### 1. 대시보드
- 진행중 프로젝트 수, 시공중 건수, 준공 예정 건수, 총 계약금액 통계
- 진행중 프로젝트 목록 테이블

### 2. 프로젝트 관리
- **CRUD**: 프로젝트 등록/수정/삭제
- **상태 관리**: 입찰중 → 계약체결 → 착공 → 시공중 → 준공서류작성 → 준공검사 → 준공완료 → 하자보증 → 완료
- **프로젝트 상세**: 기본정보, 설계내역, 기성 탭
- **필드**: 공사명, 발주처, 계약유형(종합/전문/일반/용역), 계약방식(입찰/수의계약), 계약금액, 계약기간

### 3. 발주처 관리
- **CRUD**: 발주처(지자체) 등록/수정/삭제
- **필드**: 발주처명, 지역, 담당자, 연락처, 주소
- **삭제 보호**: 프로젝트가 있는 발주처는 삭제 불가

### 4. 설계내역 임포트
- **엑셀 자동 파싱**: 설계내역서 엑셀 파일을 업로드하면 자동으로 항목 추출
- **헤더 자동 탐지**: '항목명', '수량', '단가' 등 키워드로 헤더 행 자동 탐색
- **원가 구분 자동 추정**: 공종명/항목명에서 재료비/노무비/경비 자동 분류
- **보호**: 기성이 진행된 프로젝트는 설계내역 재임포트 차단

### 5. 기성처리
- **회차 자동 생성**: 설계내역 기반으로 기성 상세 자동 생성
- **전회 누계 자동 이월**: 이전 회차의 누계가 다음 회차의 전회로 자동 설정
- **진도율 입력**: 공종별 금회 진도율 입력 → 금액 자동 산출
- **자동 계산**: 전회/금회/누계 금액·비율 자동. 회차 총 기성금액 자동 합산
- **100% 초과 방지**: 누계 진도율이 100%를 초과하면 에러
- **상태 관리**: 작성중 → 청구완료 → 승인완료 / 보완요청

### 6. 기성내역서 엑셀 내보내기
- **양식**: 공사명, 발주처, 도급금액, 기성일자 헤더 포함
- **내역**: 공종별 설계금액, 전회/금회/누계(비율+금액) 표시
- **서식**: 셀 테두리, 배경색, 숫자포맷(#,##0), 퍼센트포맷, 병합셀
- **합계행**: 설계총액, 전회/금회/누계 합계 및 비율 자동 계산

### 7. 설정
- 보험요율 정보 표시 (2026년 기준)

## Phase 2 (예정)

### 준공서류 자동화
- 발주처별 준공서류 체크리스트 템플릿
- 준공사진첩 자동 생성 (EXIF 날짜 추출, 공종별 분류, 양식 자동 배치)

- 달력 뷰 출역 관리
- 급여 자동 계산 (4대보험, 원천징수)
- 노무비 지급대장/급여명세서 엑셀 생성
- 프로젝트별 투입 노무비 집계 (기성 연동)
