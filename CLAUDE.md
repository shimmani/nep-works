# NEP-WORKS

Electron + React 데스크톱 앱. 엔이피 업무 자동화 시스템.

- **스택**: Electron (electron-vite), TypeScript, React, Vitest
- **실행**: `source ~/.zshrc && cd nep-works && yarn dev`
- **테스트**: `source ~/.zshrc && cd nep-works && yarn test`
- **빌드**: `yarn build` / `yarn package`
- **구조**: `src/main/` (Electron main), `src/renderer/` (React UI), `templates/` (문서 템플릿)
